# Excel Sales Dashboard

Interactive Excel dashboard created as part of my data analytics learning journey.

## Project Overview
This project analyzes sales performance using an interactive Excel dashboard built with PivotTables, Form Controls, and VBA macros.

## Features
- Interactive dashboards using Form Control checkboxes
- PivotTables and PivotCharts for dynamic analysis
- KPI tracking: Total Sales, Target Hit %, Away from Target %
- City-wise and executive-wise insights
- VBA macros for dashboard switching and automation

## Tools & Skills
- Microsoft Excel (Advanced)
- PivotTables & PivotCharts
- Excel Form Controls
- VBA (Macros)
- Data Analysis & Dashboard Design
